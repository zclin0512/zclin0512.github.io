python中list列表的使用方法

列表是多个有序的序列，可以存放多种元素，列表是可变的 ## 1.创建列表 ## 创建空列表：list = [] 创建带有元素的列表 ：list1 = [13,25,16,48,59,63] ## 2.列表的访问与替换 ## 访问列表：list[索引] 替换列表内容：list[索引] = 内容 ## 3.列表的操作 ## 列表的组合：list3 = list1 + list2 列表的重复：list2 = list1 * 2 列表的截取：list[start : end],end不能超过列表范围，否则会报错，包前不包后
