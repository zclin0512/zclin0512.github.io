python中的判断，循环语句

发表于 2018-04-01 更新于 2023-08-25 分类于 python

while循环

概念 :while 语句用于循环执行程序， 即在某条件下，循环执行某段程序，以处理需要重复处理的相同任务 结构：while 判断条件: 执行语句...

在使用while循环的时候，循环体内必须有判断条件重新赋值的语句，此处不同与for循环

for循环

概念: for循环可以遍历任何序列的项目，如一个列表或者一个字符串。 格式： for 变量名 in 集合: 语句 逻辑: 按顺序取“集合”中的每个元素赋值给“变量”,在去执行语句。如此循环往复，直到取完“集合”中的元素截止

在for循环中，不需要将i值重新赋值，默认为自动循环

break 与 continue

break 语句用来终止循环语句，即当循环没有正常结束的时候直接结束循环进行循环体后边的语句 continue 语句也是用来终止循环的，不过与break不同的是，continue是跳出本次循环直接进行下一次循环，不会直接终止循环

if 语句

结构： if 判断条件1: 执行语句1…… elif 判断条件2: 执行语句2…… elif 判断条件3: 执行语句3…… else: 执行语句4…… 其中elif语句视情况而定，再python中没有switch-case语句，可以用if-elif-else来代替 在判断语句中可以灵活的运用and or 等语句来达到判断的目的
