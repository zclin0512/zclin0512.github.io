vue-cli中package.json详解

发表于 2022-03-01 更新于 2023-08-07 分类于 vue

1.简介

创建项目时会自动生成package.json文件，定义了当前项目所需要的各种模块信息，以及项目的配置信息（如名称、版本、许可证等元数据）。

npm install 可以根据当前的package.json文件来自动下载所需模块。

npm init 可以自动生成package.json文件， 根据当前项目情况。

2.详解

示例

1. vue-cli-service 命令

  • vue-cli-service serve 启动开发服务器

  • vue-cli-service build 打包

    其他参数

    • -modern 使用现代模式构建应用，为现代浏览器交付原生支持的 ES2015 代码，并生成一个兼容老浏览器的包用来自动回退。
    • -target 允许你将项目中的任何组件以一个库或 Web Components 组件的方式进行构建。更多细节请查阅构建目标。
    • -report 和 -report-json 会根据构建统计生成报告，它会帮助你分析包中包含的模块们的大小。
