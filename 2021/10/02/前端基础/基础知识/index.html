html基础知识

1、url（Uniform Resource Locator）

统一资源定位器，它是WWW的统一资源定位标志，就是指网络地址。

URL由三部分组成：资源类型、存放资源的主机域名、资源文件名。

也可认为由4部分组成：协议、主机、端口、路径

URL的一般语法格式为：

(带方括号[]的为可选项)：

protocol :// hostname[:port] / path / [;parameters][?query]#fragment

2、网页

  1. 网站与网页

    网页是构成网站的基本元素。

    • 网站是指因特网上根据一定的规则，使用HTML等制作的用于展示特定内容的相关网页集合
    • 网页是其中的一页，通常是HTML格式的文件，主要通过浏览器来阅读。

  2. 网页是构成网站的基本元素，通常由图片、链接、文字、声音、视频等元素构成，以html后缀，也称为html文件。

  3. html指的是超文本标记语言（HyperText Markup Language）描述网页的一种语言，不是编程语言，而是标记语言。使用“标记”来注明文本、图片和其他内容，以便在web浏览器中显示。

  4. 超文本含义

    • 可以加入图片，声音，动画，多媒体能内容（超越了文本的限制）
    • 可以从一个文件跳转到另一个文件，与世界各地主机的文件链接（超级链接文本）

3、浏览器

浏览器是网页显示，运行的平台。

浏览器内核（渲染引擎）：负责读取网页内容，整理讯息，计算网页的显示方式并显示页面

  • IE

    内核：Trident备注： IE，猎豹安全，360极速，百度浏览器

  • firefox

    内核：Gecko 备注： 火狐浏览器

  • sifari 内核

    webkit 备注： 苹果浏览器内核

  • chrome 内核

    blink 备注： chrome/Opera浏览器内核，blink为webkit的分支

4、web标准

web不标准是由W3C组织和其他标准化组织指定的一系列标准的集合。W3C（万维网联盟）是国际最著名的标准化组织。

为什么需要web标准

  1. 让web的发展前景更广阔
  2. 内容能被更广泛的设备访问
  3. 更容易被搜索引擎搜索
  4. 降低网站流量费用
  5. 是网站更易于维护
  6. 提高网页的浏览速度

web标准的构成

主要包括 结构，表现，和行为三个方面。

  • 结构：结构用于对网页元素进行整理和分类
  • 表现：表现用于设置网页元素的板式、颜色、大小等外观样式
  • 行为：行为是指网页模型的定义及交互的编写

语法规范

1、基本语法概述

  1. HTML标签是由尖括号包围的关键词。
  2. HTML标签通常是成对出现的，被称为双标签。第一个中的标签是开始标签，第二个标签是结束标签。
  3. 有写特殊的标签必须是单个标签，称为单标签。

2、标签关系

双标签关系可分为两类：包含关系和并列关系

3、文档类型声明标签

文档类型声明，告诉浏览器用哪种HTML版本显示

4、字符集

字符集是多个字符的集合，以便计算机能够识别和存储各种文字。在 标签内，可以通过

标签的charset属性来规定HTML文档应使用哪种字符编码。

常用值有GB2312，BIG5，GBK和UTF-8。
